## <img src="https://user-images.githubusercontent.com/74038190/212257468-1e9a91f1-b626-4baa-b15d-5c385dfa7ed2.gif" width="26"> The Problem

Simple to-do lists collapse the moment work has **stages and priorities**. Teams (and students juggling projects) need to see what's queued, what's moving, and what's done — at a glance.

## 💡 The Solution

TaskFlow organizes work into dynamic stage columns with live priority tagging. Everything updates instantly through clean centralized state — **zero page reloads**.

## <img src="https://user-images.githubusercontent.com/74038190/212257465-7ce8d493-cac5-494e-982a-5a9deb852c4b.gif" width="26"> Features

- 📌 **Dynamic Kanban columns** — To Do → In Progress → Done
- 🏷️ **Live priority tagging** — High / Medium / Low with visual indicators
- 🔄 **Centralized state** — move a card, every view syncs instantly
- 🏃 **Sprint-style tracking** — progress visible per stage
- 🎨 **Professional dark dashboard** — responsive across devices

## <img src="https://user-images.githubusercontent.com/74038190/213910845-af37a709-8995-40d6-be59-724526e3c3d7.gif" width="26"> Engineering Notes

- **Data modeled before UI:** boards → columns → tasks hierarchy designed first; components fell out naturally
- State shape keeps column consistency guaranteed when tasks move between stages
- Priority colors tuned for dark-theme readability
