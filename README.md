# 🚀 TaskFlow Enterprise — Agile Kanban Workflow Suite

An enterprise-inspired Kanban task management suite engineered for modern software development teams. Features live state updates, multi-stage column transitions, and priority indicators.

## ✨ Highlights
- 📌 Dynamic workflow column tracking (To Do, In Progress, Done)
- ⚡ Priority tagging (High, Medium, Low) & real-time re-ordering
- 🎨 Responsive dark-themed modern dashboard

## 🛠️ Tech Stack
- **Framework:** React 19 + Vite
- **Architecture:** Clean State Management & Modern CSS
